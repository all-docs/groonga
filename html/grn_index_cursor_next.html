grn_index_cursor_next

__attribute__((dllimport)) grn_posting *grn_index_cursor_next(grn_ctx *ctx, grn_obj *index_cursor, grn_id *term_id)

Retrieve the current posting information and move the cursor to the the next.

ctx
The context object.
index_cursor
Target cursor.
term_id
The buffer to store the ID of the current target record of cursor. If `NULL` is specified, it will not be stored.

Returns grn_posting if the next posting is found, `NULL` otherwise. You don't need to free the returned grn_posting.
