grn_table_cursor_next

__attribute__((dllimport)) grn_id grn_table_cursor_next(grn_ctx *ctx, grn_table_cursor *tc)

Move the cursur forward to the next and return its record ID. Return GRN_ID_NIL when the end is reached

ctx
The context object
tc
Target cursor

Returns Record ID on there is a next, GRN_ID_NIL on no next or on error
