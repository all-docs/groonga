grn_ctx_open

__attribute__((dllimport)) grn_ctx *grn_ctx_open(int flags)

Open the context.

This function also allocates the context itself. However, grn_ctx_init requires the caller to allocate a context itself.

The context opened with this function should be closed with grn_ctx_close.

flags
Initialization options.

Returns Initialized context object, `NULL` if initialization fails.
