grn_proc_create

__attribute__((dllimport)) grn_obj *grn_proc_create(grn_ctx *ctx, const char *name, int name_size, grn_proc_type type, grn_proc_func *init, grn_proc_func *next, grn_proc_func *fin, unsigned int nvars, grn_expr_var *vars)

Create the procedure in the database handled by `ctx`.

ctx
The context object.
name
Name of procedure.
name_size
The number of bytes of `name` parameter. If negative value is specified, `name` parameter is assumed that NULL-terminated string.
type
Type of procedure.
init
Initialization callback function.
next
Main callback function.
fin
Finalization callback function.
nvars
The number of `vars`.
vars
Variables used in callback functions.

Returns The procedure object on success, `NULL` on error.
