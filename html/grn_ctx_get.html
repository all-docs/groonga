grn_ctx_get

__attribute__((dllimport)) grn_obj *grn_ctx_get(grn_ctx *ctx, const char *name, int name_size)

Retrieve the object corresponding to the specified name from the database that the context is using.

ctx
The context object.
name
The name of the object to be retrieved.
name_size
The size of the `name` in bytes.

Returns grn_obj if the object that has the specified name is found, `NULL` otherwise.
