grn_table_delete_by_id

__attribute__((dllimport)) grn_rc grn_table_delete_by_id(grn_ctx *ctx, grn_obj *table, grn_id id)

Delete the record matching the ID in the table.

ctx
The context object
table
The table
id
ID of record to be deleted

Returns GRN_SUCCESS on success, the appropriate grn_rc on error. For example, GRN_INVALID_ARGUMENT is returned if the record does not exist.
