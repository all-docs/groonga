grn_obj_delete_hook

__attribute__((dllimport)) grn_rc grn_obj_delete_hook(grn_ctx *ctx, grn_obj *obj, grn_hook_entry entry, int offset)

Delete the hook set on the object.

ctx
The context object.
obj
The target object.
entry
The type of hook.
offset
The offset of execution order.

Returns GRN_SUCCESS on success, the appropriate grn_rc on error. For example, GRN_INVALID_ARGUMENT is returned if there was no hook in `offset`.
