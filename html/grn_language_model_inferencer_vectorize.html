grn_language_model_inferencer_vectorize

__attribute__((dllimport)) grn_rc grn_language_model_inferencer_vectorize(grn_ctx *ctx, grn_language_model_inferencer *inferencer, const char *text, int64_t text_length, grn_obj *output_vector)

Vectorize a text.

In other words, compute embeddings of a text.

ctx
The context object.
inferencer
The inferencer.
text
The text to vectorize.
text_length
The byte size of `text`. You can use `-1` if `text` is a `\0`-terminated string.
output_vector
GRN_DB_FLOAT32 vector as an output. A caller must initialize this by GRN_FLOAT32_INIT and GRN_OBJ_VECTOR.

Returns GRN_SUCCESS on success, the appropriate grn_rc on error.
