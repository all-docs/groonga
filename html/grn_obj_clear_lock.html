grn_obj_clear_lock

__attribute__((dllimport)) grn_rc grn_obj_clear_lock(grn_ctx *ctx, grn_obj *obj)

Forcefully clear locks on an object.

Unlike grn_obj_unlock, which decrements the lock count by one, this function forcefully resets the lock count of an object to zero, unlocking it regardless of the current lock count.

In general you should not use this function. Forcefully clearing locks might lead to data corruption or inconsistencies within the database. Use it only when absolutely necessary and ensure that you understand the potential consequences.

ctx
The context object.
obj
The target object whose lock is to be cleared.

Returns GRN_SUCCESS on success.
