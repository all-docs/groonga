grn_encoding_to_string

__attribute__((dllimport)) const char *grn_encoding_to_string(grn_encoding encoding)

Return string representation for the encoding. For example, `grn_encoding_to_string(GRN_ENC_UTF8)` returns `"utf8"`.

encoding
The encoding

Returns String representation for the encoding on success, "unknown" on invalid encoding.
