grn_table_setoperation

__attribute__((dllimport)) grn_rc grn_table_setoperation(grn_ctx *ctx, grn_obj *table1, grn_obj *table2, grn_obj *res, grn_operator op)

Execute the set operation of `table1` and `table2` according to the specification of `op` and store the result into `res`.

The table specified in `res` will be destructed.

ctx
The context object.
table1
The table.
table2
The table.
res
Specify `table1` or `table2`.
op
Type of operation. - GRN_OP_OR - GRN_OP_AND - GRN_OP_AND_NOT - GRN_OP_ADJUST

Returns GRN_SUCCESS on success, the appropriate grn_rc on error. For example, GRN_INVALID_ARGUMENT is returned if `table1` is `NULL`.
