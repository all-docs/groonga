grn_obj_lock

__attribute__((dllimport)) grn_rc grn_obj_lock(grn_ctx *ctx, grn_obj *obj, grn_id id, int timeout)

Lock an object with a specified timeout.

You should ensure that every call to grn_obj_lock is paired with a call to grn_obj_unlock to maintain lock balance. Failing to do can lead to deadlocks.

ctx
The context object.
obj
The target object to lock.
id
The ID of the target object.
timeout
The maximum time to wait for the lock, in seconds.

Returns GRN_SUCCESS on success, the appropriate grn_rc on error. For example, GRN_RESOURCE_DEADLOCK_AVOIDED is returned if the lock could not be acquired within the specified timeout.
