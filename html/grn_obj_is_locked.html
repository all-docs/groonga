grn_obj_is_locked

__attribute__((dllimport)) unsigned int grn_obj_is_locked(grn_ctx *ctx, grn_obj *obj)

Check if an object is currently locked.

ctx
The context object.
obj
The target object whose lock status is to be checked.

Returns `0` if there are no locks. Return the number of acquired locks if the object or any of its related sub-objects are currently locked.
