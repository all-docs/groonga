grn_obj_get_value

__attribute__((dllimport)) grn_obj *grn_obj_get_value(grn_ctx *ctx, grn_obj *obj, grn_id id, grn_obj *value)

Retrieve the value of the record corresponding to the given ID in the specified object.

ctx
The context object
obj
The target object from which to retrieve the value
id
The ID of the target record
value
The buffer to store the retrieved value (must be prepared by the caller)

Returns The value of the specified record in the object.
