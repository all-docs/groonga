grn_obj_defrag

__attribute__((dllimport)) int grn_obj_defrag(grn_ctx *ctx, grn_obj *obj, int threshold)

Defragment an object to reduce fragmentation in the database file.

The defragmentation process aims to reduce the fragmentation in the database file area occupied by the object, based on the provided threshold value.

See `ctx->rc` for error details.

ctx
The context object.
obj
The target object to defragment.
threshold
The threshold value used to guide the defragmentation process.

Returns The total number of segments on which defragmentation was performed.
