grn_obj_set_info

__attribute__((dllimport)) grn_rc grn_obj_set_info(grn_ctx *ctx, grn_obj *obj, grn_info_type type, grn_obj *value)

Set information on the object.

ctx
The context object.
obj
The target object.
type
The type of information.
value
The value to set.

Returns GRN_SUCCESS on success, the appropriate grn_rc on error. For example, GRN_INVALID_ARGUMENT is returned if `obj` is `NULL`.
