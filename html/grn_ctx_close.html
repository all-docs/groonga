grn_ctx_close

__attribute__((dllimport)) grn_rc grn_ctx_close(grn_ctx *ctx)

Close the context.

This function releases memory that was managed by the context and closes. (Internally, grn_ctx_fin is called to release allocated memory.)

This function closes the context opened by grn_ctx_open. If the context is initialized with grn_ctx_init, it needs to be closed with grn_ctx_fin.

ctx
The context object initialized by grn_ctx_open.

Returns GRN_SUCCESS on success, the appropriate grn_rc on error.
