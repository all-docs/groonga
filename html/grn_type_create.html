grn_type_create

__attribute__((dllimport)) grn_obj *grn_type_create(grn_ctx *ctx, const char *name, unsigned int name_size, grn_obj_flags flags, unsigned int size)

Define a new type in DB

ctx
The context object
name
Name of type to create
name_size
Length of the name of type to be created
flags
GRN_OBJ_KEY_VAR_SIZE, GRN_OBJ_KEY_FLOAT, GRN_OBJ_KEY_INT, GRN_OBJ_KEY_UINT or GRN_OBJ_KEY_GEO_POINT
size
Maximum length if GRN_OBJ_KEY_VAR_SIZE, otherwise length (in bytes)

Returns A newly created type on success, `NULL` on error
