grn_table_update

__attribute__((dllimport)) grn_rc grn_table_update(grn_ctx *ctx, grn_obj *table, const void *src_key, unsigned int src_key_size, const void *dest_key, unsigned int dest_key_size)

Change the key of the record matching the `src_key` of the table. Specify the new key and its byte length in `dest_key` and `dest_key_size`. This operation is allowed only for table of type GRN_TABLE_DAT_KEY

ctx
The context object
table
Target table
src_key
Key of record to be updated
src_key_size
Length of `src_key` (byte)
dest_key
New key
dest_key_size
Length of `dest_key_size` (byte)

Returns GRN_SUCCESS on success, the appropriate grn_rc on error. For example, GRN_OPERATION_NOT_PERMITTED is returned if not GRN_TABLE_DAT_KEY.
