grn_ctx_at

__attribute__((dllimport)) grn_obj *grn_ctx_at(grn_ctx *ctx, grn_id id)

Retrieve the object corresponding to the specified ID from the context or the database that the context is using.

ctx
The context object.
id
The object ID to be retrieved.

Returns grn_obj if the object that has the specified ID is found, `NULL` otherwise.
