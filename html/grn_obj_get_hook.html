grn_obj_get_hook

__attribute__((dllimport)) grn_obj *grn_obj_get_hook(grn_ctx *ctx, grn_obj *obj, grn_hook_entry entry, int offset, grn_obj *data)

Get the hook set on the object. If hook data is set, it is stored in `data`.

ctx
The context object.
obj
The target object.
entry
The type of hook.
offset
The offset of execution order.
data
The buffer to store hook data.

Returns The procedure object if the hook is set, `NULL` otherwise.
