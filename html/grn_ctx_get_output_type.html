grn_ctx_get_output_type

__attribute__((dllimport)) grn_content_type grn_ctx_get_output_type(grn_ctx *ctx)

Get the current output type of the context.

Normally, you don't need to call this function unless you need to check the output type explicitly.

ctx
The context object.

Returns Return the output type of the context, which is one of the values defined in grn_content_type. Return GRN_CONTENT_NONE if the context is invalid.
