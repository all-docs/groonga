grn_table_lcp_search

__attribute__((dllimport)) grn_id grn_table_lcp_search(grn_ctx *ctx, grn_obj *table, const void *key, unsigned int key_size)

Execute longest common prefix (LCP) search and return ID found.

Execute longest common prefix search if the table is GRN_TABLE_PAT_KEY or GRN_TABLE_DAT_KEY. If the table is GRN_TABLE_HASH_KEY, search by exact match.

ctx
The context object.
table
The table.
key
Search key.
key_size
Length of `key`.

Returns ID on success, GRN_ID_NIL on not found or error.
