grn_obj_set_finalizer

__attribute__((dllimport)) grn_rc grn_obj_set_finalizer(grn_ctx *ctx, grn_obj *obj, grn_proc_func *func)

Set the callback function when finalizing the object.

ctx
The context object.
obj
Target object. Table, table cursor, column, procedure, and expression can be specified.
func
Callback function when finalizing.

Returns GRN_SUCCESS on success, GRN_INVALID_ARGUMENT if `obj` is not table, table cursor, column, procedure, or expression.
