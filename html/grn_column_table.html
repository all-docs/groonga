grn_column_table

__attribute__((dllimport)) grn_obj *grn_column_table(grn_ctx *ctx, grn_obj *column)

Retrieve a table to which a given column belongs.

ctx
The context object.
column
The target column to retrieve the table to which it belongs.

Returns The table object to which the target column belongs on success, `NULL` if the column is broken.
