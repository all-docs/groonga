grn_encoding_parse

__attribute__((dllimport)) grn_encoding grn_encoding_parse(const char *name)

Parse encoding name and return grn_encoding. For example, `grn_encoding_parse("UTF8")` returns GRN_ENC_UTF8.

name
The encoding name

Returns grn_encoding matching `name` on success, GRN_ENC_UTF8 on invalid encoding name.
