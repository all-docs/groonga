grn_table_get_key

__attribute__((dllimport)) int grn_table_get_key(grn_ctx *ctx, grn_obj *table, grn_id id, void *keybuf, int buf_size)

Get the key assigned to the ID. If the size of the key found is larger than `buf_size`, it is not stored in `keybuf`.

ctx
The context object.
table
The table.
id
The ID to be found.
keybuf
Buffer to store the record key.
buf_size
Size of `keybuf` in bytes.

Returns Key size of the record on success, `0` on not existed.
