grn_obj_get_range

__attribute__((dllimport)) grn_id grn_obj_get_range(grn_ctx *ctx, grn_obj *obj)

Retrieve the value range based on the ID associated with an object.

For example, it may return GRN_DB_INT32 from grn_builtin_type to indicate that the object operates within the integer range.

ctx
The context object.
obj
The target object whose range ID is to be returned.

Returns The ID of the range object. Returns GRN_ID_NIL if no range is associated or the object is `NULL`.
