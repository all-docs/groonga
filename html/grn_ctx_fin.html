grn_ctx_fin

__attribute__((dllimport)) grn_rc grn_ctx_fin(grn_ctx *ctx)

Finish the context.

This function releases memory that was managed by the context and finishes.

This function finishes the context initialized by grn_ctx_init. If the context is opened with grn_ctx_open, it need to be closed with grn_ctx_close.

ctx
The context object initialized by grn_ctx_init.

Returns GRN_SUCCESS on success, the appropriate grn_rc on error.
