grn_ctx_use

__attribute__((dllimport)) grn_rc grn_ctx_use(grn_ctx *ctx, grn_obj *db)

Set the database that the context is using.

You can initialize it if you specify `NULL` for the parameter `db`.

Don't use it with context that has GRN_CTX_PER_DB flag.

ctx
The context object.
db
The database object.

Returns GRN_SUCCESS on success, the appropriate grn_rc on error.
