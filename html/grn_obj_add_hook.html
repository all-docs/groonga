grn_obj_add_hook

__attribute__((dllimport)) grn_rc grn_obj_add_hook(grn_ctx *ctx, grn_obj *obj, grn_hook_entry entry, int offset, grn_obj *proc, grn_obj *data)

Add a hook to the object. If multiple hooks are set for the object, they are called in the order they are registered in the hook list.

ctx
The context object.
obj
The target object.
entry
The type of hook.
offset
The offset of execution order. Add to the beginning if `0` is specified. Add to the end if `-1` is specified.
proc
The procedure object.
data
The hook data.

Returns GRN_SUCCESS on success, the appropriate grn_rc on error.
