grn_table_at

__attribute__((dllimport)) grn_id grn_table_at(grn_ctx *ctx, grn_obj *table, grn_id id)

Search the table for a record of ID and return the specified ID if it exists, or GRN_ID_NIL if it does not

It is costly to perform. Do not call them frequently

ctx
The context object
table
The table or database
id
The ID to be found

Returns Specified ID on success, GRN_ID_NIL on not found or error
