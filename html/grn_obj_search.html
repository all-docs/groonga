grn_obj_search

__attribute__((dllimport)) grn_rc grn_obj_search(grn_ctx *ctx, grn_obj *obj, grn_obj *query, grn_obj *res, grn_operator op, grn_search_optarg *optarg)

Search for `obj` in `query`.

ctx
The context object.
obj
The object to be searched.
query
Search query.
res
The table to store results.
op
Logical operation of the table stored in `res` and the table of the search result. - GRN_OP_OR - GRN_OP_AND - GRN_OP_AND_NOT - GRN_OP_ADJUST
optarg
Search options. For example, you can specify the search mode in grn_search_optarg::mode. If GRN_OP_EQUAL is specified for grn_search_optarg::mode, it searching for equal record.

Returns GRN_SUCCESS on success, the appropriate grn_rc on error. For example, GRN_INVALID_ARGUMENT is returned if `obj` is `NULL`.
