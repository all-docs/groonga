grn_obj_rename

__attribute__((dllimport)) grn_rc grn_obj_rename(grn_ctx *ctx, grn_obj *obj, const char *name, unsigned int name_size)

Rename a persistent object in the database used by the context.

This function updates the name of the specified object (`obj`) to the new name provided. The object must be a persistent object.

ctx
The context object.
obj
The target object to be renamed.
name
The new name for the given object.
name_size
The size of the `name` in bytes.

Returns GRN_SUCCESS on success, the appropriate grn_rc on error.
