grn_language_model_loader_load

__attribute__((dllimport)) grn_language_model *grn_language_model_loader_load(grn_ctx *ctx, grn_language_model_loader *loader)

Load a language model.

If the target model is already loaded, this reuses the model instead of loading a new model.

See `ctx->rc` for error details.

ctx
The context object.
loader
The loader.

Returns A loaded grn_language_model on success, `NULL` on error.
