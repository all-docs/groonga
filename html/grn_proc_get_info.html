grn_proc_get_info

__attribute__((dllimport)) grn_obj *grn_proc_get_info(grn_ctx *ctx, grn_user_data *user_data, grn_expr_var **vars, unsigned int *nvars, grn_obj **caller)

Get information on procedure and expression that use `user_data`.

ctx
The context object.
user_data
The user data passed to grn_proc_func.
vars
The buffer to store defined variables.
nvars
The buffer to store the number of `vars`.
caller
The buffer to store calling expression.

Returns The procedure object. It may be `NULL`.
